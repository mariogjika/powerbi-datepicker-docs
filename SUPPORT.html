Support & Usage — Date Picker Slicer

Guidance for adding the visual to a report, configuring fields, and troubleshooting common issues.

Add the visual

  1. Power BI Desktop → Get more visuals → Import a visual from file.
  2. Select the .pbiviz file and add the visual to the canvas.

Bind a Date field

  1. In the Fields pane, drag a date column into the visual’s Date field well.
  2. The visual will automatically use the bound date field to filter the report.
Tip: Use a clean Date column (not text) for the best results.

Choose a selection mode

  • Use the dropdown to pick a preset such as Custom range or Exact date.
  • Open Format → Selection options to enable more presets (rolling, fiscal, ISO weeks).

Tooltips

  • Hover over the visual to see a summary of the current selection.
  • Add measures to the Tooltips field well to display values in the tooltip.
  • Range selections show From, To, and the range length.

Troubleshooting

  • No data or blank visual: Ensure a date field is bound in the Date field well.
  • Selection doesn’t apply: Check Format → Behavior for Apply vs Auto-apply.
  • Unexpected dates: Verify your data type is Date/DateTime and the report is refreshed.
  • Tooltip missing measures: Add measures to the Tooltips field well.
