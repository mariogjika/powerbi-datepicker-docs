Privacy Policy — Date Picker Slicer

Power BI custom visual: Date Picker Slicer

Overview

Date Picker Slicer processes report data locally inside Power BI to render the UI and apply date filters. It does not send data to external services, and it does not store data outside the report.

Data processed

  • Date fields bound to the visual.
  • Measures added to the Tooltips field well (if any).
  • Formatting settings selected by the report author.

Data collection and transmission

The visual does not collect telemetry and does not transmit report data outside Power BI. No data leaves the user’s tenant or device via this visual.

Data storage and retention

The visual stores no data outside the report itself. Selections are stored only in Power BI’s report state (filters or bookmarks) managed by the Power BI host.
